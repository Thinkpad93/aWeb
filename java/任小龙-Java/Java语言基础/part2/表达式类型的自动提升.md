##### 表达式类型的自动提升

```java

public class HelloWorld {
    public static void main(String[] args) {
        // 123 int类型
        // 1.1F float类型
        // 99L long类型
        // 3.14 double类型
        double d1 = 123 + 1.1F + 99L + 3.14;
        //结果输出为double类型
        System.put.println(d1);

        //所有的byte、short、char类型被自动提升到int类型
        System.put.println('A' + 1); // 65 + 1 = 66
    }
}

```
