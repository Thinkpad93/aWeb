##### 位运算符

```java

public class HelloWorld {
    public static void main(String[] args) {

        /*
          3的二进制 00000011
          5的二进制 00000101

          &:
                00000011
                00000101
            -------------------------------
                00000001

          |:   
                00000011
                00000101
            -------------------------------
                00000111 = 1的4次方 +　１的2次方 + 1的1次方

          ^: 相同则为0，不同则为1
                00000011
                00000101
            -------------------------------
                00000110                                
        */
        System.out.println(3 & 5); // 1
        System.out.println(3 | 5); // 7
        System.out.println(3 ^ 5); // 6
    }
}

```
