##### 输出九九乘法表

```java

public class HelloWorld {
    public static void main(String[] args) {
        for (int line = 1; line <= 9; line++) {
            for (int i = 1; i <= line; i++) {
                System.out.print(i + "*" + line + "=" + (i * line) + "\t");
            }
            System.out.println();
        }
    }
}

```
