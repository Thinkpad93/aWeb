##### 嵌套循环

```java

public class HelloWorld {
    public static void main(String[] args) {
        /*
            若外层循环次数是M次
            内层循环次数是N次
            那么内层循环的次数则需要 M * N 次
        */
        for (int count = 1; count <= 4; count ++) {
            for (int i = 1; i <= 100; i++) {
                 System.out.println(i);
            }
            System.out.println(count);
        }
    }
}

```
