#####  Object类和常用方法 

 Object类是Java语言的根类

```java
class ABC {} 
//等价于
class ABC extends Object {}
```

为什么Object类是所有类的根类，到底什么是Object？

Object本身是对象的意思，我们发现所有的对象都具有一些共同的行为，所以，我们抽象出一个类: Object，表示对象类，其它都会继承于Object类，也就拥有Object类的方法。

引用数据类型：类/接口/数组，引用类型又称为对象类，所谓的数组变量名称，应该指数组对象。
