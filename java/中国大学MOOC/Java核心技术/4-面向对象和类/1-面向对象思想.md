类/对象例子

```java

public class Father {
    private int money = 100; // 私有
    long mobile = 13999999999L;

    public void hello() {
        System.out.println("hello");
    }
}

public class Son extends Father {
    public void hi() {
        // 子类可以扩展自己的成员方法
        System.out.println("hi~~~~~~");
    }

    public static void main(String[] arg) {
        Son s = new Son();
        System.out.println(s.mobile);
        s.hello();
        s.hi();
    }
}





```

面向对象语言主要特点
- 识认性
- 类别性
- 多态性
- 继承性
