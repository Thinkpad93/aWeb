Arrays: 处理对象是数组

- 排序，对数组排序，sort/parallelSort
- 查找，从数组中查看一个元素，binarySearch
- 批量拷贝，从源数组批量复制元素到目标数组，copyOf
- 批量赋值，对数组进行批量赋值，fill
- 等价性比较，判定两个数组内容是否相同，equals
