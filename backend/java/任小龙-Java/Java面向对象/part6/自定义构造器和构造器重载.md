##### 自定义构造器和构造器重载 

```java
class Person {
    String name = null; //人的名称
    
    //自定义构造器
    Person(String n) {
        //初始化操作
        name = n;
        System.out.printLn(name);
    }
}

class PersonDemo {
    public static void main(String[] args) {
        //创建对象，其实是在调用构造器
        //表示调用Person类中，带有一个String类型的参数
        Person p = new Person("Jack");
        System.out.printLn(p.name);
    }
}
```
