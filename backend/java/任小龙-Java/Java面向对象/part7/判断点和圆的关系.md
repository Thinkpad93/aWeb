##### 判断点和圆的关系

判断一个点和圆的关系（圆内，圆外）

```java
// 点对象
class Point {
    private int x; // 横坐标
    private int y; // 纵坐标

    // 构造器方法
    Point(int x, int y) {
        this.x = x;
        this.y = y;
    }

    public int getX() {
        return x;
    }
    public int getY() {
        return y;
    }
}
// 圆对象
class Cirele {
    private int r; // 半径

    Cirele(int r) {
        this.r = r;
    }
    /*
      判断点和圆的关系
      返回：
      	1.表示在圆外
      	0.表示在圆周上
      	-1.表示在圆内
    */
    int jugde(Point p) {
        int xxyy = p.getX() * p.getX() + p.getY * p.getY();
        // 当前圆半径的平方
        int rr = this.r * this.r;
        if (xxyy > rr) {
            return 1;
        } else if (xxyy < rr) {
            return -1;
        } else {
            return 0;
        }
    }
}

class PointDemo {
    public static void main(String[] ags) {
        // 创建一个点对象
        Point p = new Point(3, 4);
        // 创建一个半径为5的圆对象
        Cirele c = new Cirele(5);

        // 判断
        int ret = c.jugde(p);

        System.out.printLn(ret);
    }
}
```
