##### 输出三角形图案

```java

public class HelloWorld {
    public static void main(String[] args) {
        /*
            输出如下图案
            *
            **
            ***
            ****
            *****
        */
        for (int line = 1; line <= 5; line ++) {
            for (int i = 1; i <= line; i++) {
                System.out.println("*");
            }
            System.out.println();
        }
    }
}

```
