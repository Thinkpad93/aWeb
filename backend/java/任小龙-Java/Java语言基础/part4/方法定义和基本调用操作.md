> 方法定义和基本调用操作

```java

public class HelloWorld {
    public static void main(String[] args) {
        /*
          方法必须要调用才能生效
          如果方法使用了static修饰符
          则可以使用类的名称访问方法
          如果方法有返回值，必须使用return语句
        */
        System.out.println("main");
        HelloWorld.dowork(); // 调用静态方法dowork
    }
    static void dowork() {
      System.out.println("dowork");
    }
}
```
