> 方法的执行流程分析

```java

public class HelloWorld {
    public static void main(String[] args) {
      System.out.println("main");
      // 在调用方法中所传入的参数叫实参
      int num = HelloWorld.getSums(1, 2);
      System.out.println(num);
    }
    // 在方法定义中所传入的参数叫形参
    static int getSums(int a, int b) {
      // 最终返回 a + b 的结果
      return a + b;
    }
}
```
