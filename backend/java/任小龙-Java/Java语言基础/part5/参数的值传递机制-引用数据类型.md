##### 方法参数的值传递机制之引用数据类型

```java

class ParamaterDemo {
    public static void main(String[] args) {
        int[] arr = new init[]{10, 99};
        swap(arr); //交换第一个和最后一个元素的值
    }
    static void swap(int[] arr) {
        int temp = arr[0];
        arr[0] = arr[arr.length - 1];
        arr[arr.length - 1] = temp;
    }
}

```
