##### Java5 对数组的支持-增强 for 循环

```java

class ForeachDemo {
    public static void main(String[] args) {
        int[] nums = new int[] {10,20,30,40,50};
        //增强for循环
        for (int ele : nums) {
            System.out.println(ele);
        }
        System.out.println("Hello world");
    }
}

```
