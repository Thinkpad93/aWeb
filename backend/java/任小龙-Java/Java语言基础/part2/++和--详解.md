##### ++和--详解

```java

public class HelloWorld {
    public static void main(String[] args) {
        //前置和后置的区别
        //前置: 表示对加1之后的结果进行运算
        //后置: 表示对加1之前的值(原始值)进行运算
        int a1 = 5;
        int b1 = ++ a1; // a1 + 1
        System.out.println("a1 = " + a1 + "b1 =" + b1); // a1 = 6,b1 = 6
        System.out.println("==================");

        int a2 = 5;
        int b2 = a2 ++; // a2 = (5) + 1
        System.out.println("a2 = " + a2 + "b2 =" + b2); // a2 = 6,b2 = 5
    }
}

```
