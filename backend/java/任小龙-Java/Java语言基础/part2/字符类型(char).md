##### 字符类型(char)

```java

public class HelloWorld {
    public static void main(String[] args) {
        char c1 = 'A'; //直接表示符号
        System.put.println(c1);
        char c2 = 65; //直接表示符号对应的10进制值
        System.put.println(c2);
        char c3 = '\u0041'; //表示16进制的值
        System.put.println(c3);
    }
}

```
