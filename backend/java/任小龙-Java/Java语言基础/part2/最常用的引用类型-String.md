##### 最常用的引用类型-String

```java

public class StringDemo {
    public static void main(String[] args) {
        // String的常量
        System.out.println("Hello");
        // String变量
        String name = "龙";
        System.out.println("name=" + name);
        // 输出 "Hello"
        System.out.println("\"Hello\"");
        // 字符串和任意类型相连接，结果都是字符串类型
        System.out.println(name + 7 + 8);
    }
}

```
