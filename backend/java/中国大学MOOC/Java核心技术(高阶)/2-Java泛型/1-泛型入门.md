泛型 Generic Programming
编写的代码可以被很多不同类型的对象所重用

```java

ArrayList<String> list = new ArrayList<String>();
list.add('123');
list.add('456');
list.add('789');

String s = list.get(1);

ArrayList<Integer> list2 = new ArrayList<>();
list2.add(123);
list2.add(456);
list2.add(789);

int i = list2.get(1);

```

- 泛型类
  ArrayList HashSet HashMap

- 泛型方法
  Collections.binarySearch Arrays.sort 等

- 泛型接口
  List Iterator 等
