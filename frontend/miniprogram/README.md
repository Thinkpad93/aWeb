#### dpr & dpi & ppi
dpr 设备像素比，物理像素/设备独立像素=dpr，一般以iphone6的dpr为准，dpr=2
ppi 一英寸显示屏上的像素点个数
dpi 最早指的是打印机在单位面积上的打印墨点数，墨点越多越清晰
