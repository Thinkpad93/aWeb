#### 关于移动端 ios 时间格式显示问题
```js
let str = '2020-04-15 12:30:59';
// 时间戳，但是在ios上会报错，问题出在ios不支持斜杠格式的日期
new Date(str).getTime();
// 解决办法
let freedomTime = str.replace(/-/g, '/'); // '2020/04/15 12:30:59'
new Date(freedomTime).getTime();
```
